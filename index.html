02. Me, Myself and I

Hello! My name is Alex and I'm a curiosity-driven Junior Front-End Developer who enjoy creating things that live on the internet. I'm passionate about bringing both the technical and visual aspects of digital products to life.

My interest in web development started back in 2021 when I was a Store Manager in a Tennis Shop. While I was using their e-commerce website daily, I realised that it's been made really bad, so I decided that it's about time to start learning some skills. Since I didn't had any background in Computer Science, I took a Web Developer Course for beginners because I had to become familiar with the programming languages. At the end of the year I decided to take a JavaScript Course as well, because I was curious to learn more about it.

I'm a fast learner, constantly trying to improve my technical abilities while providing good and clean code at the same time. I'm happiest when I'm creating, learning, exploring and thinking about how to make things better.
